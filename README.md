# Titan1280
Python program to run ICI Titan 1280 camera. Works on Mac, but modification of the line contianing AVFoundation and swapping for platform correct webcam interface should work on PC and Linux


![Alt text](https://github.com/zevhoo/Titan1280/blob/main/Titan1280%20Screenshot.png "Optional title")
